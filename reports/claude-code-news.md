# Claude Code 最新情報まとめ（2026年4月時点）

## 🔧 最新バージョンアップデート（v2.1.87〜v2.1.92）

### v2.1.92（4月4日）
- **`forceRemoteSettingsRefresh`ポリシー設定**を追加：リモート管理設定の取得が失敗するとCLIが起動をブロック（フェイルクローズ方式）
- **Bedrockセットアップウィザード**をインタラクティブに追加：ログイン画面から「サードパーティプラットフォーム」選択時にAWS認証・リージョン設定・資格情報確認・モデルピン留めを案内
- `/cost`コマンドにモデル別・キャッシュヒット別の内訳表示を追加（サブスクリプションユーザー向け）
- `/release-notes`がインタラクティブなバージョン選択UIに変更
- リモートコントロールセッション名がホスト名をデフォルトプレフィックスとして使用（例：`myhost-graceful-unicorn`）

### v2.1.91（4月2日）
- **MCPツール結果の最大サイズ**をアノテーション指定で最大500Kまで拡張（DBスキーマなど大きなデータの切り捨てを回避）
- **`disableSkillShellExecution`設定**を追加：スキル・カスタムスラッシュコマンド・プラグインコマンドでのインラインシェル実行を無効化
- `claude-cli://open?q=`ディープリンクでマルチラインプロンプトをサポート
- プラグインが`bin/`配下に実行ファイルを同梱し、Bashツールからベアコマンドとして呼び出し可能に
- `--resume`時に会話履歴が失われるバグを修正
- `cmd+delete`がiTerm2・kitty・WezTerm等で行頭削除されない問題を修正

### v2.1.90（4月1日）
- **`/powerup`コマンド**を追加：アニメーションデモ付きでClaude Codeの機能を学べるインタラクティブレッスン
- `CLAUDE_CODE_PLUGIN_KEEP_MARKETPLACE_ON_FAILURE`環境変数を追加：オフライン環境でのプラグインキャッシュ保持
- レート制限ダイアログが繰り返し開いてクラッシュする無限ループバグを修正
- `--resume`時のプロンプトキャッシュミス問題を修正（v2.1.69以降のリグレッション）
- PostToolUseフックによるファイル保存後に`Edit`/`Write`が失敗するバグを修正

### v2.1.89（4月1日）
- **`PreToolUse`フックに`"defer"`権限決定**を追加：ヘッドレスセッションでツール呼び出しを一時停止・再開可能
- **`CLAUDE_CODE_NO_FLICKER=1`**環境変数でフリッカーのないalt-screenレンダリングをサポート
- **`PermissionDenied`フック**を追加：自動モードの拒否後に`{retry: true}`でリトライ可能
- `@`メンションの候補に名前付きサブエージェントを追加
- `MCP_CONNECTION_NONBLOCKING=true`で`-p`モード時のMCP接続待機をスキップ可能

### v2.1.87（3月29日）
- Cowork Dispatchでメッセージが配信されないバグを修正
